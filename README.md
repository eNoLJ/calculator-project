# calculator-project

### 주요 기능
#### 1. 사칙연산 계산
사용자가 입력한 두 숫자와 연산기호(+, -, x, /)를 통해 계산을 수행합니다.
- 제네릭을 사용해 Integer, Double 등 다양한 타입 지원
- 0으로 나누기 시 예외 처리

#### 2. 연산 결과 이력 저장
계산 결과를 history(List\<T\>)에 저장합니다
- 히스토리 조회 가능
- 첫 번째 결과 삭제 기능

#### 3. 특정 값보다 큰 결과 조회
사용자가 입력한 값보다 더 큰 게산결과만 필터링하여 출력합니다.

### 실행 방법
1. 프로젝트를 다운로드 또는 클론
2. IDE(Intellij 등)에서 열기
3. Main 클래스 실행
4. 메뉴 선택 후 계산 진행

### 실행 흐름 예시
````
1. 계산하기
2. 연산 결과 이력 보기
3. 첫 번째 연산 결과 삭제
4. 입력값보다 큰 결과값 출력
5. 종료
입력: 1

첫 번째 숫자를 입력하세요: 3.5
두 번째 숫자를 입력해주세요: 2
사칙연산 기호를 입력하세요: x

결과: 7.0
````
